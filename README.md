# LZ-ANI
Experimental ANI calculation based on entropy

Ta gałąź zawiera wersję z początku 2023, w której dodano nową wersję trybu all2all, tzn. z nowym zrównolegleniem oraz opcjonalne filtrowanie z użyciem wyjścia z Kmer-db.

Tu są też inne stare rzeczy, które póki co nie będą rozwijane.

Od tego momentu w bieżącej wersji nieco okroimy funkcjonalność, głównie pod kątem projektu z klastrowaniem wirusów.
